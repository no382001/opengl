Instructions


The primary requirement for running the tutorials is a video card that supports at least OpenGL-3.3. The code provided by the tutorials was developed on Ubuntu Linux using NetBeans as a primary development platform. You should be able to compile it without NetBeans by simply running 'make' from the command line. The following libraries/apps are required (all of them available via apt-get):
  1. g++
  2. freeglut3-dev
  3. libglew1.5-dev

The code was tested only on my machine that has the following details:

CPU: Intel Core 2 Extreme QX9770
OS: Ubuntu Linux 11.04 64 bits
Video card: GIGABYTE NVIDIA GeForce GTX550 TI
Driver version: 275.21

Please let me know of any problems to compile/run the tutorials on other platforms (fixes will be greatly appreciated!).
